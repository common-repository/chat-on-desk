A plugin for sending whatsapp notification after placing orders using WooCommerce

== Description ==

This is a WooCommerce add-on. By Using this plugin admin and buyer can get notification about their order via whatsapp using Chat On Desk.

The WooCommerce Order Whatsapp Notification plugin for WordPress is very useful, when you want to get notified via whatsapp after placing an order. Buyer and seller both can get whatsapp notification after an order is placed. whatsapp notification options can be customized in the admin panel very easily.

PS: This plugin requires an account(paid/demo) with [www.chatondesk.com](https://www.chatondesk.com) service to send notifications. Demo account comes with some free credits to let you test it out.
